## Descripción
Este repositorio contiene el archivo fuente (`.msapp`) de la aplicación desarrollada para mi Trabajo de Fin de Grado en Ingeniería Informática. La aplicación permite [objetivo principal de la app].

## Contenido del repositorio
* `ProvesFísiques.msapp`: Archivo de la aplicación listo para ser importado en Power Apps.
* `ResultatsProvesFísiques.msapp`: Archivo de la aplicación listo para ser importado en Power Apps.

## Cómo ejecutar la aplicación
1. Descarga el archivo `ProvesFísiques.msapp` de este repositorio.
2. Accede a [Power Apps Studio](https://make.powerapps.com).
3. Haz clic en **Abrir** > **Examinar** y selecciona el archivo descargado.
4. **Importante:** Asegúrate de configurar las conexiones a [Excel/SharePoint/Dataverse] descritas en la memoria del TFG para que los datos carguen correctamente.
